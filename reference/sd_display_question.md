# Create a placeholder for a reactive survey question

This function is depreciated - use
[`sd_output()`](https://pkg.surveydown.org/reference/sd_output.md)
instead.

## Usage

``` r
sd_display_question(id)
```

## Arguments

- id:

  A unique identifier for the question.

## Value

A 'shiny' UI element that serves as a placeholder for the reactive
question.
